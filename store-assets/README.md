# Chrome Web Store Assets

Place your store listing assets in this folder.

## Required Assets

### Screenshots (Required - at least 1)
- **Dimensions**: 1280x800 or 640x400 pixels
- **Format**: PNG or JPEG
- **Naming**: `screenshot-1.png`, `screenshot-2.png`, etc.

**Suggested screenshots:**
1. Extension button visible on Claude.ai with chat input
2. Side panel showing a rephrased prompt result
3. Settings/options page with provider selection

**How to capture (macOS):**
1. Resize Chrome window to exactly 1280x800
2. Navigate to a supported AI site (e.g., claude.ai)
3. Trigger the extension to show it in action
4. Use `Cmd+Shift+4` to capture, or DevTools `Cmd+Shift+P` > "Capture screenshot"
